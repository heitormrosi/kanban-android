# 📋 Projeto Kanban - Android

## 📖 Descrição
Este é um projeto de Quadro Kanban desenvolvido como exercício prático para a disciplina de Dispositivos Móveis II. O objetivo é gerenciar tarefas de forma visual e intuitiva, permitindo a organização de fluxos de trabalho.

## 🎓 Contexto
O projeto foi realizado dentro do curso de Informática para a Internet (Integrado ao Ensino Médio) no IFES - Campus Serra. 🏛️

## 🛠️ Tecnologias Utilizadas

* Linguagem: Kotlin 💜
* IDE: Android Studio 🤖

## ✅ Funcionalidades
* 📝 Criação de novas tarefas.

* 📑 Organização por colunas (A fazer, Em andamento, Concluído).

* 📱 Interface nativa e responsiva.
